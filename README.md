#### an app which lets you find a perfect dish to test your cooking skills.
#### you can choose by *Category*, by *Region* or by *Ingredient*.


main logic was coded in three days.\
the hardest part was finding an API which provides proper data to create something meaningful within the requirements and constraints of the task.\
after looking through tons of different free APIs a decision was made to use [themealdb API](https://www.themealdb.com/api.php) for its simplicity and *three* types of filters.
